Introduction

  • Item Name : Min&Max Variations
  • Item Version : v1.2.3
  • Author : Alejandro E. Rendon
  • License : GPLv3

Min&Max Variations for WooCommerce plugin allow you to setup limits and display different customizations in the product screens.

Features

Fields to set the limits for each variation in the Product edition section.

Product view customizations.

  1. Price for each unit.
  2. Variable price calculated using the Price by Unit x Quantity.
  3. Quantity selector input:
    1. Block and hide if the variation is not selected.
    2. Validations for quantity using the fields with limit information (Min/Max quantity).

How to Install

Upload via WordPress Admin (Recommended)

  1. Download the .zip file.
  2. Open de administrator menu Plugins > Add new.
  3. Click the Upload Plugin button. The action will show the section to upload a file.
  4. Use the Choose file button and select the .zip file and after press the Install Now button.
  5. Activate Min&Max Variations for WooCommerce from Plugins page

Upload manually

This method is not recommended because the initial setup could be corrupted or not created

  1. Unzip the downloaded .zip file.
  2. Upload the Min&Max Variations for WooCommerce folder into the wp-content/plugins/ directory of your WordPress site.
  3. Activate Min&Max Variations for WooCommerce from Plugins page

Remove/uninstall

Just in case you want to deactivate and/or remove the plugin, you can do it through the wordpress admin. Here is a video with instructions.
