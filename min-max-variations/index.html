Min&Max Variations for WooCommerce plugin allow you to setup limits and display different customizations in the product screens.

Requirements

You will need the following dependencies/conditions to customize this plugin.

  1. Versión de WordPress 5.0 or superior
  2. Active Woocomerce plugin
